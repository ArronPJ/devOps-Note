# Install for macOS

## Homebrew
- [official Website](https://brew.sh/)

### Install Homebrew
- This command copy from official website

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```

### CLI Usage Sample

```
brew info azure-cli
brew update
brew install azure-cli
```
